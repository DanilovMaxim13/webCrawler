Описание:
----------------------
Программа предназначенная для перебора страниц Интернета с целью
занесения информации о них в базу данных компьютера.
----------------------

Требования
•Python версия не ниже 3.6
•модуль requests версия 2.21.0
•модуль beautifulsoup4 версия 4.7.1

Использование:
 В интерактивном режиме:
--------
     >>>import start_program
--------
     >>>start_program.crawl(save_path, *url)
--------

 Через консоль:
      python start_program.py --url *url, --path path

Где url - это ссылки с которых вы хотите начать работу краулера,
а save_path - это путь куда вы хотите сохранять HTML файлы страниц.

Подробности реализации:
----------------------
Модуль отвечающий за начало игры называется start_program.py, в нем 
присутствует метод crawl() который и нужно запускать для начала краулинга.
В модуле Crewler хронится основная часть программы.
В модуле modul_test.py хранятся тесты на краулер.На данный момент
покрытие этих тестов составляет 89%.
